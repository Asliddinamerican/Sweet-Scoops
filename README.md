# 🍦 Sweet Scoops

**Har bir lahza uchun mazali muzqaymoq!**

Sweet Scoops — bu muzqaymoq do'koni uchun yaratilgan zamonaviy, chiroyli va responsive landing page. Loyiha HTML, CSS va Bootstrap 5 yordamida qurilgan bo'lib, mijozlarga eng mashhur muzqaymoq lazzatlarini jozibador tarzda taqdim etadi.

---

## ✨ Xususiyatlari

- 🧭 **Responsive Navbar** — mobil qurilmalarda ham qulay ishlaydigan navigatsiya paneli
- 🍨 **Featured Flavors** — Klubnika, Shokolad, Mango va Vanil kabi mashhur ta'mlarni ko'rsatuvchi kartochkalar
- 💰 **Narxlar va "Buy Now" tugmalari** — har bir mahsulot uchun narx va xarid tugmasi
- 🎁 **Aksiya banneri** — "2 ta oling, 1 tasi bepul" kabi diqqatni tortuvchi taklif
- 📍 **Contact bo'limi** — manzil, telefon, email va ish vaqti haqida ma'lumot
- 🎨 **Custom soyalar (box-shadow)** — navbar va header uchun maxsus CSS effektlari
- 📱 **To'liq responsive dizayn** — Bootstrap grid va utility klasslari asosida

---
